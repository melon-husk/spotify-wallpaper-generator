# Spotify Wallpaper Generator

You can this web app to generate custom wallpaper from your Spotify Playlist.

## Installation

```bash
git clone https://github.com/melon-husk/spotify-wallpaper-generator

npm install
```

## Usage

```bash
npm run dev
```
